# Secluso

</div>

<p align="center">
  Private home security that is easy to set up and easy to trust
</p>

Secluso is a home security camera system made for everyday people. It runs on a Raspberry Pi with a camera and a simple mobile app. Your videos stay end to end encrypted so you control who can see them.

You get fast alerts, live view, and recordings without handing over your footage. The camera encrypts video before it leaves your home. The server only relays encrypted data.

You can use your own Raspberry Pi with any compatible camera and do not need our plug and play device. If you want a ready made option, we are building a dedicated camera based on our Raspberry Pi prototype. Learn more at https://secluso.com and join the mailing list for updates.

## Get started

Download the Secluso setup tool app from https://github.com/secluso/secluso/releases. It guides you through setup, builds the Raspberry Pi image, and sets up the server for you with no technical knowledge needed.

## Security note

Secluso assumes the server is fully untrusted and still keeps your videos encrypted end-to-end. For a detailed discussion, see SECURITY.md.
